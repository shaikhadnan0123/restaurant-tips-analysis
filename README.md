# Python Intro Milestone Portfolio Project

## Overview
This project was created after completing the Python milestone in Artificial Intelligence and Machine Learning. It demonstrates practical use of Python Intro, Python, Comments, Input Function, Conditional Statements, Type Of Operators, Type Conversion, Strings, String Operations, Lists, Tuples, Loops, Dictionaries, Functions, Lamda Functions, Map(), Filter(), And Reduce() Functions, File I/o, Oops in a portfolio-ready format.

## Objectives
- Define the business problem
- Prepare and analyze the dataset
- Present useful insights and recommendations
- Publish a clean, reviewable GitHub repository
